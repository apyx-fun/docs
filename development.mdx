## Preview locally

- **Node:** 19+
- From the repo root: `cd docs && mint dev`
- Open `http://localhost:3000`; edits hot-reload.

Custom port: `mint dev --port 3333`

## Useful commands

- **Broken links:** `mint broken-links`
- **Update CLI:** `npm i -g mint` (or `npm mint update` if your CLI supports it)
