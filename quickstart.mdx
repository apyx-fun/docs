## Run the docs locally

1. Install Mintlify CLI: `npm i -g mint`
2. From the `docs` folder: `mint dev`
3. Open `http://localhost:3000` — preview updates as you edit.

## Where the code lives

- **Contract (Solana program)** — `apyx-contract/` — Anchor; instructions: launch, buy, sell, resolve_duel, claim_mercy, migrate.
- **SDK** — `apyx-sdk/` — IDL and client helpers for the program.
- **Relayer** — `apyx-relayer/` — Watches for duel resolution conditions and submits `resolve_duel` (and optional auto-claim); cannot change pairing or winner.
- **Frontend / admin** — `apyx-client/`, `apyx-admin/` — Apps that use the SDK.

For full contract account and instruction details, see the README and docs in `apyx-contract/docs/`.
